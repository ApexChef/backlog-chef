## Implementation Details

Tech Lead: Here's how I'd build this:

**1. Loading Indicator Library**
```typescript
// src/display/chef-loader.ts
class ChefLoader {
  constructor(
    private step: string,
    private style: LoadingStyle,
    private showTips: boolean
  ) {}

  start(): void {
    this.showRandomVerb();
    if (this.showTips) {
      this.startTipRotation();
    }
  }

  private showRandomVerb(): void {
    const verbs = COOKING_VERBS[this.step];
    const verb = verbs[Math.floor(Math.random() * verbs.length)];
    this.display(verb);
  }

  private startTipRotation(): void {
    setInterval(() => {
      const tip = this.getContextualTip();
      this.displayTip(tip);
    }, this.config.tip_rotation_seconds * 1000);
  }
}
```

## ACCEPTANCE CRITERIA

**Chef-Themed Loading:**
1. All long operations (>2s) show chef-themed loading indicators
2. Each pipeline step has appropriate cooking terminology
3. Verbs rotate randomly within step's theme
4. Loading style configurable (spinner, progress, steps, time)
5. Emoji support for visual appeal
6. Graceful fallback if terminal doesn't support Unicode

**Tips System:**
7. Tips displayed during operations longer than 5 seconds
8. Tips rotate every 5-10 seconds (configurable)
9. Tips are contextual when possible (step-specific)
10. Three tip categories: chef wisdom, CLI usage, chef quotes
11. Can disable tips with config flag
12. Tips database easily extensible

**Configuration:**
13. `~/.backlog-chef/config.yaml` controls display settings
14. `chef_theme: false` disables for professional mode
15. `show_tips: false` disables tips
16. `tip_rotation_seconds` configurable
17. `loading_style` selectable (spinner/progress/steps/time)

**User Experience:**
18. Loading indicators work in all terminal types
19. No flickering or performance issues
20. Tips are helpful, not annoying
21. Chef metaphors are clear and relatable
22. CLI tips are actionable and accurate
23. Quotes are properly attributed

**Technical:**
24. Loading library doesn't block execution
25. Tips load from JSON/YAML file
26. Supports custom user-contributed tips
27. Works offline (no external tip fetching)
28. Minimal performance overhead (< 50ms)

## Risks

**Risk 1: Annoyance Factor**
- Too many cooking references might annoy some users
- Mitigation: Easy to disable (`chef_theme: false`)

**Risk 2: Terminal Compatibility**
- Emojis don't work on all terminals
- Mitigation: Auto-detect and fallback to text

**Risk 3: Performance**
- Loading indicators could slow down operations
- Mitigation: Async display, minimal overhead

**Risk 4: Cultural Sensitivity**
- Cooking references may not translate well
- Mitigation: Keep metaphors universal, support i18n later
